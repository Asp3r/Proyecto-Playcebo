Sobre Mí

Soy un apasionado del desarrollo web con una sólida formación full-stack y experiencia en la creación de proyectos completos. Actualmente, estoy usando mis habilidades como freelancer para costear mi vida como estudiante universitario, lo que me ha permitido combinar mi pasión por la tecnología con mi compromiso académico.
Durante mi tiempo en la secundaria, participé en la Olimpiada Matemática Argentina (OMA), donde fui galardonado dos años consecutivos. Esta experiencia me enseñó a solucionar problemas complejos y a trabajar bajo presión, habilidades que aplico en cada proyecto que desarrollo. Mi objetivo es siempre ofrecer soluciones eficientes y bien diseñadas que superen las expectativas de mis clientes.

Proyectos Front-End

Barbería Alura

Una página web responsive y profesional para una barbería ficticia creada como un proyecto de integración para el curso "Desarrollo Front-End" de Alura Latam, con el objetivo de poner en práctica todos los conocimientos teóricos adquiridos y demostrar una comprensión sólida de los conceptos básicos de HTML5 y CSS3.

Number Guesser

Un juego web sencillo donde el usuario tiene que adivinar el número generado por la máquina, usando las pistas que la misma le proporciona luego de cada intento fallido. Hecho con funcionalidades básicas de HTML5 y CSS3, así como JavaScript para hacer el sitio interactivo y realizar las operaciones lógicas necesarias para el correcto funcionamiento del juego.

Proyectos Back-End

Foro-Hub

Foro-Hub es una API REST que realiza operaciones CRUD sobre la información de publicaciones creadas en una red social ficticia llamada 'Foro-Hub'. Es un proyecto integrador para el curso 'Desarrollador Back-End con Spring Boot' de Alura ONE (un programa educativo creado en alianza entre Alura Latam y Oracle ONE).

Proyecto de integración para la formación en Back-End de Alura ONE. Un programa de consola que almacena de forma duradera información sobre libros y autores en una base de datos, y permite al usuario acceder a esta información posteriormente. Todo esto implementando la API de Gutendex.
